# GojekSlicing
 Slicing Home pada aplikasi gojek menggunakan flutter. Aplikasi ini merupakan slicing dari tutorial yang saya dapatkan di youtube dan saya dapat mengimplmentasikannya dengan baik. Slicing disini menggunakan komponen yang lengkap dari asset, component, hingga ke view semua ada.
